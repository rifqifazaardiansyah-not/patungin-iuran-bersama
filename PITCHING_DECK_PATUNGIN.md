**Iuran Digital, Transparan & Terorganisir**

## SLIDE 2: PROBLEM STATEMENT
**Durasi: 1.5 menit**

### Masalah Spesifik di Pasar:

#### 🔴 **Problem #1: Transparansi Rendah = Konflik Internal**
- **68% bendahara** masih menggunakan Excel + WhatsApp
- **41% organisasi** mengalami konflik internal terkait transparansi keuangan
- **Rp 2.8 Miliar** dana hilang/tidak jelas pertanggungjawabannya (2023)

#### 🔴 **Problem #2: Administrasi Manual = Waktu Terbuang**
- Bendahara menghabiskan **8-12 jam/minggu** untuk:
  - Mencatat pembayaran manual
  - Mengingatkan anggota satu per satu via WhatsApp
  - Membuat laporan keuangan di Excel
- **40% waktu** bendahara = administrasi, bukan strategi

#### 🔴 **Problem #3: Koleksi Iuran Tidak Efisien**
- **23% anggota** telat bayar karena lupa/tidak diingatkan
- **15 hari rata-rata** waktu pengumpulan iuran penuh
- **Rp 500rb - 2 juta** potensi loss per organisasi/tahun akibat iuran tidak tertagih

### Data Pendukung:
- **7.2 juta mahasiswa** di Indonesia (Dikti 2024)
- **500+ organisasi kampus** aktif per universitas
- **Rp 50.000 - 500.000** iuran per anggota per semester
- **Total Addressable Market: Rp 180 Miliar/tahun** (estimasi konservatif)

## SLIDE 3: SOLUSI
**Durasi: 2 menit**

### **Patungin: Platform SaaS untuk Manajemen Iuran Digital**

#### Produk yang Ditawarkan:
1. **Dashboard Digital** - Kelola iuran organisasi & event dalam 1 platform
2. **Multi-Payment Gateway** - Bank Transfer, QRIS, E-Wallet
3. **Auto Reminder** - WhatsApp & Email otomatis untuk penagihan
4. **Real-Time Reporting** - Transparansi penuh untuk semua anggota
5. **Mobile-First Design** - Akses dari smartphone, kapan saja

### Bagaimana Kami Mengatasi Masalah?

| Masalah | Solusi Patungin | Impact |
|---------|-----------------|--------|
| **Transparansi rendah** | Dashboard real-time + laporan otomatis | ✅ 100% transparansi, 0 konflik |
| **Administrasi manual** | Otomasi penuh: pembayaran, reminder, laporan | ✅ Hemat 10 jam/minggu (83% efisiensi) |
| **Koleksi tidak efisien** | Reminder otomatis + multi-payment | ✅ 95% collection rate dalam 7 hari |

### User Journey:

**Bendahara:**
1. Buat event/iuran baru (2 menit)
2. Set target anggota & payment method
3. Sistem auto-send reminder ke anggota
4. Monitor dashboard real-time
5. Export laporan PDF/Excel (1 klik)

**Anggota:**
1. Terima notifikasi reminder WhatsApp
2. Lihat detail iuran di app
3. Bayar via Bank/QRIS/E-Wallet
4. Upload bukti pembayaran
5. Status langsung terupdate

**ROI untuk User:**
- **10 jam/minggu** waktu admin dihemat = **Rp 500.000/bulan** (@ Rp 50k/jam)
- **23% late payment** berkurang → **95% collection rate**
- **100% transparansi** → **0 konflik** internal
